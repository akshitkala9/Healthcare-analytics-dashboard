# Healthcare Analytics Dashboard

## Overview
An interactive Power BI dashboard built to analyze healthcare data and provide insights into patient demographics, billing trends, insurance coverage, and medical conditions.

## Objectives
- Analyze patient records and healthcare performance.
- Monitor billing and insurance-related metrics.
- Identify trends in admissions and medical conditions.

## Technologies Used
- Power BI
- Excel / CSV
- DAX
- Data Cleaning & Transformation

## Dashboard Features
- Interactive Filters & Slicers
- Patient Demographics Analysis
- Billing Trend Visualization
- Insurance Provider Comparison
